O trabalho consiste na implementação de uma árvore Treap(Tree+Heap), estruturas de dados balanceadas em que os nós contém uma chave e uma prioridade de inserção.
Entrada:
Insira numero de operacoes, que pode ser:
	-insercao (chave a inserir) (prioridade) -> para inserir um numero na arvore balanciada
	-busca (chave a buscar)                  ->buscar um numero na arvore
	-remocao (chave a remover)               -> remover uma chave da arvore
	imprissao modo -> modos podem ser: ordem; posordem; preordem e largura

Saida:
imprime  as  linhas  correspondentes  para  cada  operação se necessario. 
	-inserção,  se  a  chave  já  existir  deverá  ser  impresso  a  frase  ”Elemento  ja  existente”;
	-remoção,  se a chave não existe, imprime ”Chave nao localizada”.
	-cada nó é impresso como uma tupla, logo, no formato (chave, prioridade).

Os arquivos .in podem ser utilizados como exemplo de input, enquando os arquivos .out são exemplos de output
