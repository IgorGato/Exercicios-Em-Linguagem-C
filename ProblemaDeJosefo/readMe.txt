Implementação de um algoritmo que calcula o sobrevivente no problema teorico de contagem "O problema de Josefo"
Para mais informações acesse: https://en.wikipedia.org/wiki/Josephus_problem
  
Entrada:
	A entrada começa com um inteiro T(<= 200), denotando o número de casos de teste. 
	Cada caso contém dois positivos inteiros n(1 <= n <= 10000) e k(1 <= k <= 10000).

Saida:
 	
	Para cada caso, imprima o número do caso e a posição da pessoa que sobreviveu.


Os arquivos .in podem ser utilizados como exemplo de input, enquando os arquivos .out são exemplos de output
